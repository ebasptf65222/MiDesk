# 免费发布 Electron 项目方案

## 现状分析

- 项目已配置 `electron-builder` publish → GitHub Releases (`XiaomiMiMo/MiMo-Code`)
- 项目已配置 `electron-updater` 自动更新（检查/下载/安装）
- 代码托管在 Gitee (`chang-zhi-qiang/mi-desk`)，但 publish 目标是 GitHub
- 所有 package scripts 都是 `--publish never`，从未自动发布过

## 推荐方案：GitHub Releases（完全免费）

**原理**：GitHub Releases 提供免费的文件托管（单文件 1GB，仓库总容量 10GB），配合 `electron-updater` 实现自动更新，零成本。

### 步骤 1：推送代码到 GitHub

```bash
# 添加 GitHub remote（如果还没添加）
git remote add origin https://github.com/XiaomiMiMo/MiMo-Code.git

# 推送代码
git push -u origin main
```

> 确保 GitHub 仓库已存在且有推送权限。如果仓库是 private，GitHub Releases 仍可免费使用（公开下载需要 GitHub 账号）。

### 步骤 2：生成 GitHub Personal Access Token

1. 去 GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. 创建 token，权限选择：
   - Repository access: 选择 `MiMo-Code` 仓库
   - Permissions: Contents → Read and Write（用于上传 release 资产）
3. 保存 token 备用

### 步骤 3：本地构建并发布

```bash
# 设置 GitHub token 环境变量
# Windows PowerShell:
$env:GH_TOKEN = "你的token"

# 构建并发布到 GitHub Releases
npm run build
npx electron-builder --win --publish always
```

这会：
1. 构建 Windows 安装包（NSIS + Portable）
2. 创建 GitHub Release（tag = 版本号）
3. 上传安装包到 Release Assets

### 步骤 4：自动更新验证

已有的 `src/main/updater.ts` 会自动：
- 启动时检查 GitHub Releases 是否有新版本
- 有新版本时通知用户下载
- 下载完成后提示安装

无需修改任何代码，现有自动更新配置已对接 GitHub Releases。

## 备选方案对比

| 方案 | 费用 | 难度 | 适合场景 |
|------|------|------|----------|
| **GitHub Releases** | 免费 | 低（已配置好） | ✅ 最佳选择 |
| Gitee Releases | 免费 | 中（需额外配置） | 国内分发 |
| Cloudflare R2 | 免费 10GB | 高（需自建更新服务） | 大文件/CDN |
| Netlify/Vercel | 免费带宽 | 高（需自建更新服务） | 仅限 web 下载页 |

## 注意事项

1. **macOS 签名**：如果要分发 macOS 版本，Apple Developer 账号（$99/年）是必须的。Windows 代码签名也需付费证书。不签名的话用户会看到安全警告，但仍可安装使用。
2. **Linux**：AppImage 可直接分发，无需签名。
3. **仓库容量**：GitHub 单个 Release 限 1GB，每个仓库最多 10GB 总 Release 资产。对于桌面应用通常足够。
4. **隐私**：如果仓库是 public 的，任何人都能下载你的应用。如果是 private，下载时需要 GitHub 登录。

## 执行计划

不需要修改任何代码文件。只需：

1. 确认 GitHub 仓库权限就绪
2. 生成并设置 GH_TOKEN
3. 修改 package.json 的 publish scripts 为 `--publish always`（可选，也可以手动执行）
4. 运行构建发布命令
5. 验证 Release 页面有安装包
6. 验证自动更新功能正常
