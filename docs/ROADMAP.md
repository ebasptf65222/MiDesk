## 缺失功能分析

### 🔴 核心体验缺失（高优先级）

#### T1: 设置中心 - API Key 与模型配置
**问题**: 用户无法配置 API Key 和选择 AI 模型
**方案**:
- 设置页面：输入 API Key、选择模型（mimo-auto, mimo-v2.5-pro 等）
- 配置持久化到 `~/.mimo-desktop/config.json`
- 首次使用引导流程

**设计要点**:
- 输入框带眼睛图标切换显示/隐藏
- 模型选择用下拉框 + 简短说明
- 测试连接按钮验证 Key 有效性

#### T2: 会话管理 - 保存与恢复对话
**问题**: 关闭应用后对话历史丢失
**方案**:
- 对话列表侧边栏，显示历史会话
- 点击恢复之前的对话
- 支持重命名、删除会话
- 自动保存当前对话

**设计要点**:
- 左侧新增「会话列表」面板
- 最新会话置顶
- 搜索框快速查找

#### T3: 斜杠命令系统
**问题**: 终端的 `/` 命令在桌面端不可用
**方案**:
- 支持 `/help`、`/clear`、`/model`、`/session` 等命令
- 输入框输入 `/` 弹出命令列表
- 命令自动补全
- 命令历史记录

**设计要点**:
- 输入 `/` 显示下拉菜单
- 上下键选择，回车执行
- 常用命令置顶
- 命令说明提示

| 命令 | 功能 |
|------|------|
| `/help` | 显示帮助 |
| `/clear` | 清空当前对话 |
| `/model` | 切换模型 |
| `/session` | 查看/切换会话 |
| `/export` | 导出对话 |
| `/stats` | 查看 token 统计 |

#### T4: 多工作区支持
**问题**: 只能同时打开一个项目
**方案**:
- 顶部工作区切换器
- 支持同时打开多个文件夹
- 每个工作区独立的对话和终端

**设计要点**:
- 下拉菜单切换工作区
- 工作区标签显示项目名
- 关闭工作区保留会话

### 🟡 开发效率提升（中优先级）

#### T4: 全局搜索与替换
**问题**: 无法在项目中搜索代码
**方案**:
- `Ctrl+F` 当前文件搜索
- `Ctrl+Shift+F` 全项目搜索
- 正则表达式支持
- 搜索结果高亮 + 跳转

**设计要点**:
- 搜索面板固定在顶部
- 结果列表显示文件名 + 行号 + 上下文
- 替换支持单个/全部替换

#### T5: Git 集成
**问题**: 无法查看代码变更和提交
**方案**:
- 显示当前分支名
- 变更文件列表（M/A/D 标记）
- 一键提交（输入 commit message）
- 查看 diff

**设计要点**:
- 底部状态栏显示分支
- 侧边栏新增「源代码管理」面板
- 提交按钮 + 快捷键

#### T6: 快捷键系统
**问题**: 缺少常用 IDE 快捷键
**方案**:
- `Ctrl+S` 保存文件
- `Ctrl+P` 文件快速打开
- `Ctrl+Shift+P` 命令面板
- `Ctrl+`` 切换终端
- `Ctrl+B` 切换侧边栏

**设计要点**:
- 快捷键提示显示在菜单旁
- 支持自定义快捷键
- 冲突检测

---

### 🟢 体验优化（低优先级）

#### T7: 状态栏信息
**问题**: 缺少文件状态和位置信息
**方案**:
- 显示当前文件路径
- 行号/列号
- 编码格式（UTF-8）
- 换行符（CRLF/LF）
- Git 分支名

**设计要点**:
- 固定在窗口底部
- 点击可快速切换选项
- 不遮挡内容

#### T8: 主题系统
**问题**: 无法切换明暗主题
**方案**:
- 深色主题（默认）
- 浅色主题
- 跟随系统
- 自定义主题色

**设计要点**:
- 设置中一键切换
- 实时预览
- 记住用户选择

#### T9: 通知系统
**问题**: 操作反馈不明确
**方案**:
- Toast 通知（保存成功、错误提示）
- 进度条（文件保存中、AI 生成中）
- 声音提示（可选）

**设计要点**:
- 右上角弹出
- 3 秒自动消失
- 支持手动关闭

#### T10: 自动保存
**问题**: 忘记保存导致代码丢失
**方案**:
- 可配置自动保存间隔（30秒/1分钟/5分钟）
- 失焦时自动保存
- 保存前显示预览

**设计要点**:
- 设置中开启/关闭
- 状态栏显示保存状态
- 未保存标记（标签页圆点）

#### T11: 命令面板
**问题**: 功能入口分散
**方案**:
- `Ctrl+Shift+P` 打开
- 模糊搜索命令
- 显示快捷键
- 最近使用置顶

**设计要点**:
- 居中弹窗
- 输入框 + 结果列表
- 上下键选择，回车执行

#### T12: 多光标编辑
**问题**: 缺少高级编辑功能
**方案**:
- `Alt+Click` 添加光标
- `Ctrl+D` 选中相同词
- 列选择模式

**设计要点**:
- 光标动画高亮
- 多光标同步输入
- 撤销支持
